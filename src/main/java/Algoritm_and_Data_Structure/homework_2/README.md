## Разворот односвязного списка

Этот код реализует метод `reverse()`, который переворачивает связанный список. 
Инструкция по использованию платформы
- `Node` - это класс, представляющий узел связанного списка. У каждого узла есть ссылка на следующий узел.
- `head` - это переменная, хранящая ссылку на первый узел списка.

В классе Answer реализуйте метод reverse, который принимал бы на вход односвязный список и разворачивал бы его.В методе `reverse()` выполняется переворот списка:
- Создается три переменные: `prev`, `current` и `next`. 
- `prev` инициализируется как `null`, `current` получает значение `head`.
- Далее, в цикле `while`, выполняются следующие действия:
    - `next` получает ссылку на следующий узел после `current`.
    - `current.next` устанавливается в `prev`, тем самым разворачивая связь на предыдущий узел.
    - Затем, `prev` становится `current`, а `current` становится `next`.
    - Цикл продолжается, пока `current` не станет `null`.
- После завершения цикла, `head` устанавливается в последний узел, т.е. в `prev`.
